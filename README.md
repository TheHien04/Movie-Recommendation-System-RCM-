## 📖 Overview
This project is a **Movie Recommendation System** built as part of the **Machine Learning** course.  
It provides **personalized movie suggestions** through an **interactive chatbot interface**, allowing users to express their preferences naturally.  

The system combines **Content-Based Filtering**, **Collaborative Filtering**, and **Hybrid Recommendation Techniques** to deliver accurate and dynamic recommendations.

---

## 🧩 System Workflow

1. **User Interaction** – Chatbot collects movie preferences.  
2. **Data Processing** – Extracts keywords, genres, and ratings from user input.  
3. **Recommendation Engine** – Applies ML algorithms:
   - Content-Based Filtering  
   - Collaborative Filtering  
   - Hybrid Approach  
4. **Output** – Displays ranked movie suggestions in the chatbot.
